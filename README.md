# dynamic-nft
### Dynamic Nfts thta change based on market price. 

Followed Alchemy's tutorial. 


### Technologies used:

- Chainlink Keepers
- Chainlink vrf (v2)
- Chainlink aggregatorV3Interface (price feeds)
- Brownie framework
- Rinkeby testnet
- Hosted IPFS node
- Openzepplin contracts
